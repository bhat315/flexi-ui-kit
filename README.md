# Frontend Assignment

A simple React-based frontend project featuring **dynamic forms** and **data tables**. Built with **Vite**, **Tailwind CSS**, and **Axios** for API integration.

---

## 🛠 Technologies Used

- **React** – UI library
- **Vite** – Build tool
- **Tailwind CSS** – Styling
- **Axios** – API calls
- **React Router** – Routing (if applicable)
- **Zustand** – State management (optional)

---

## 📁 Project Structure

```
frontend-assignment/
├─ src/
│  ├─ components/
│  │  ├─ InputForm.jsx
│  │  ├─ DataTable.jsx
│  ├─ App.jsx
│  ├─ index.jsx
│  └─ index.css
├─ package.json
├─ vite.config.js
├─ tailwind.config.js
└─ postcss.config.js
```

> Only keep components that are needed for the assignment (InputForm, DataTable). Storybook files are optional and can be removed.

---

## 🚀 Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run development server

```bash
npm run dev
```

- Open [http://localhost:5173](http://localhost:5173) in your browser to see the app.

### 3. Build for production (optional)

```bash
npm run build
```

### 4. Preview production build (optional)

```bash
npm run preview
```
